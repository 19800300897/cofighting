# cofighting (全民抗疫)
### 功能描述  
基于以下疫情中的痛点：  
（1）及时寻找并隔离确诊病例最近的密切接触者以及间接接触者；   
（2）准确病例活动轨迹对比进行社会排查；  
（3）及时了解省市每日新增病例方便信息透明；  
本团队开发了一款能够监测附近感染地点，了解实时疫情状况，提醒用户外出感染风险的应用“全民抗疫”。  
### 开发环境  
本项目使用WebApp开发模式进行开，后端采用springboot框架，前端使用bootstrap框架，数据库使用MySQL，主要使用平台为PC端以及移动端。
